# 변경 로그

## v1.0.0

### 신규 기능

- Vue 3.4 + TypeScript 5 + Vite 5 기반의 코어 아키텍처
- Ant Design Vue（antdv-next）UI 컴포넌트 라이브러리 통합
- 완전한 RBAC 권한 시스템（동적 라우팅, 버튼 단위 권한, 디렉티브 권한）
- **ProTable** — 설정 중심의 고급 테이블 컴포넌트
  - 자동 검색 폼 생성
  - 13가지 열 렌더링 타입（text、date、tag、badge、money 등）
  - 7가지 검색 필드 타입
  - 테이블 헤더 필터링（키워드 검색 + 드롭다운 선택）
  - 열 너비 드래그 조정, 열 표시/숨김, 열 고정
  - 테이블 밀도 전환
- **ProForm** — 설정 중심의 고급 폼 컴포넌트
  - 20+ 필드 타입
  - 그리드 레이아웃
  - 폼 검증
  - 동적 옵션 로드
- **ProModal** — 확장 모달 컴포넌트
  - 드래그 이동
  - 엣지 리사이즈（8방향）
  - 전체 화면 전환（애니메이션 포함）
- 다국어 국제화（vue-i18n）
- 테마 시스템（6가지 프리셋 색상 + 다크 모드 + 사이드바 독립 테마）
- 멀티 탭 시스템（KeepAlive 캐시, 고정 탭, 컨텍스트 메뉴）
- Mock 데이터 시스템（faker.js + vite-plugin-mock-dev-server）
- 수직 / 수평 듀얼 레이아웃 모드
- 모바일 반응형 지원
- 글로벌 단축키 검색（Ctrl+K）
- ECharts 차트 통합
- TipTap 리치 텍스트 에디터 통합
- 21개 샘플 페이지

# 更新日志

## v1.0.0

### 新增

- 基于 Vue 3.4 + TypeScript 5 + Vite 5 的核心架构
- Ant Design Vue（antdv-next）UI 组件库集成
- 完整的 RBAC 权限系统（动态路由、按钮级权限、指令权限）
- **ProTable** — 配置化高级表格组件
  - 自动搜索表单生成
  - 13 种列渲染类型（text、date、tag、badge、money 等）
  - 7 种搜索字段类型
  - 表头筛选（关键字搜索 + 下拉选择）
  - 列拖拽调整宽度、列显隐、列固定
  - 表格密度切换
- **ProForm** — 配置化高级表单组件
  - 20+ 字段类型
  - 网格布局
  - 表单验证
  - 动态选项加载
- **ProModal** — 增强弹窗组件
  - 拖拽移动
  - 边缘缩放（8 方向）
  - 全屏切换（带动画）
- 中英文国际化（vue-i18n）
- 主题系统（6 种预设色 + 暗色模式 + 侧边栏独立主题）
- 多标签页系统（KeepAlive 缓存、固定标签、右键菜单）
- Mock 数据系统（faker.js + vite-plugin-mock-dev-server）
- 垂直 / 水平双布局模式
- 移动端响应式适配
- 全局快捷搜索（Ctrl+K）
- ECharts 图表集成
- TipTap 富文本编辑器集成
- 21 个示例页面

# 変更履歴

## v1.0.0

### 新機能

- Vue 3.4 + TypeScript 5 + Vite 5 ベースのコアアーキテクチャ
- Ant Design Vue（antdv-next）UI コンポーネントライブラリ統合
- 完全な RBAC 権限システム（動的ルーティング、ボタン単位の権限、ディレクティブ権限）
- **ProTable** — 設定駆動型の高度なテーブルコンポーネント
  - 自動検索フォーム生成
  - 13種類の列レンダリングタイプ（text、date、tag、badge、money など）
  - 7種類の検索フィールドタイプ
  - テーブルヘッダーフィルタリング（キーワード検索 + ドロップダウン選択）
  - 列幅ドラッグ調整、列表示/非表示、列固定
  - テーブル密度切り替え
- **ProForm** — 設定駆動型の高度なフォームコンポーネント
  - 20+ フィールドタイプ
  - グリッドレイアウト
  - フォーム検証
  - 動的オプション読み込み
- **ProModal** — 拡張モーダルコンポーネント
  - ドラッグ移動
  - エッジリサイズ（8方向）
  - フルスクリーン切り替え（アニメーション付き）
- 多言語国際化（vue-i18n）
- テーマシステム（6種類のプリセットカラー + ダークモード + サイドバー独立テーマ）
- マルチタブシステム（KeepAlive キャッシュ、固定タブ、コンテキストメニュー）
- Mock データシステム（faker.js + vite-plugin-mock-dev-server）
- 垂直 / 水平 デュアルレイアウトモード
- モバイルレスポンシブ対応
- グローバルショートカット検索（Ctrl+K）
- ECharts グラフ統合
- TipTap リッチテキストエディタ統合
- 21 のサンプルページ

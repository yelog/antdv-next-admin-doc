# 日志管理模块

## 概述

日志管理用于查看操作行为与系统事件，支撑审计、排障与安全分析。

- 路由：`/system/log`
- 页面：`src/views/system/log/index.vue`
- 权限：`system.log.view`
- API：`src/api/log.ts`

## 核心能力

- 操作日志查询
- 登录日志查询
- 条件筛选（用户、模块、时间）
- 导出与归档（可选）

## 实施建议

- 大数据量日志建议后端分页并限制查询窗口。
- 对关键操作日志设置更长保留期。
- 提供失败请求与异常日志的快速过滤。

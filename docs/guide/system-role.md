# 角色管理模块

## 概述

角色管理负责维护角色定义与权限集合，是 RBAC 的核心配置页面。

- 路由：`/organization/role`
- 页面：`src/views/system/role/index.vue`
- 权限：`system.role.view`
- API：`src/api/role.ts`

## 核心能力

- 角色列表分页查询
- 角色新增、编辑、删除
- 角色绑定菜单权限/按钮权限
- 角色状态管理（启用/禁用）

## 实现要点

- 角色权限分配建议使用“树 + 勾选”交互。
- 超级管理员角色建议前后端双重保护，不允许误删。
- 修改角色权限后，建议触发登录态权限刷新。

## 典型流程

1. 新建角色并填写标识。
2. 选择菜单与操作权限。
3. 保存后回到用户管理绑定角色。
4. 使用测试账号验证权限生效。

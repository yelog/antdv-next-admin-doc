# 集成导航（外链与 IFrame）

## 场景

示例模块演示了后台中常见的外部系统接入方式：内嵌 IFrame 与外链跳转。

- 路由组：`/examples/external/*`
- 页面：`src/views/examples/external/*`

## 两种接入方式

1. IFrame 内嵌：在系统内部保留统一导航体验。
2. 外链跳转：适合跳转到独立站点或文档门户。

## 配置点

- 路由 `meta.externalLink` 可声明外链地址。
- IFrame 页面建议统一白名单来源并配置安全策略。
- 外链打开方式建议区分当前页与新窗口。

## 实施建议

- 对第三方系统增加可用性兜底（超时/降级页）。
- 明确跨系统登录态策略（SSO 或独立登录）。
- 记录外链访问事件，便于审计分析。
